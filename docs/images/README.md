# スクリーンショット/画像

このディレクトリには、READMEやドキュメントで使用する画像ファイルを配置してください。

## 必要な画像

### discord-notification-sample.png
Discord通知のサンプル画像。

実際の配信通知のスクリーンショットを撮影してください。

**サイズ推奨**: 800x600px または類似のアスペクト比

**含めるべき内容**:
- Discord Webhookで送信された埋め込みメッセージ
- チャンネル名
- 配信タイトル
- サムネイル画像
- メンション（@everyone等）

## 画像の追加方法

1. スクリーンショットを撮影
2. このディレクトリに配置
3. ファイル名は `discord-notification-sample.png` にする
4. README.mdで参照されている画像パスと一致させる

## 注意事項

- 個人情報（ユーザー名、アバター等）が含まれる場合は編集してください
- APIキーやWebhook URLなどの機密情報が映り込まないよう注意してください
- ファイルサイズは1MB以下に抑えてください（GitHub表示最適化）
